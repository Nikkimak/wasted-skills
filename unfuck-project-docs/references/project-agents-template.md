# Project `AGENTS.md` Template Reference

This file defines what the project-level `AGENTS.md` must cover.
The raw reusable template body used by the skill lives at `assets/templates/project-agents-template.md`.

Use that raw template when materializing a project.
Use this reference when checking whether the template still matches the intended contract.

The project-level `AGENTS.md` should cover, at minimum:

- project summary and scope
- repository model and runtime repo relationship
- canonical read order
- source-of-truth rules
- required update rules
- runtime ownership boundaries
- deploy model
- project-specific non-negotiable boundaries
- useful project paths
- notes for agents

The raw template and this reference should stay aligned.
If the contract changes, update both in the same edit cycle.
