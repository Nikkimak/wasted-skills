# Context

Use this directory as the canonical project wiki for this project.

## Source Of Truth Rules

- `current/` = current live truth
- `decisions/` = accepted rules and boundaries
- chosen future-work area = future or in-progress design work
- `runbooks/` = procedures
- `components/` = service ownership guides
- `domains/` = helpful domain context, not stronger than accepted decisions or current truth
